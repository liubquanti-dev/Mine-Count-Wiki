Mine Count

Про сервер

Mine Count - загальнодоступний Minecraft сервер від liubquanti.

Як доєднатися?

Сервер підримує як Java, так і Bedrock видання гри.

Якщо ви граєте на Java виданні то доєднитися можна за адресою: mcount.liubquanti.click.

У випадку з Bedrock виданням вам треба використати ту саму адресу та порт 27323.

Сервер не працює?
